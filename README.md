# Evaluating Hydrological Process Information for Testing Models as Hypotheses Using a Virtual Reality Model 
Repository for the used data.

Folder "RawData" contains rds. data of perturbed `observations' as used for the evaluating in R.

Folder "ThresholdFata" contains data with information about the necessary threshold relaxations for each of the 1800 simulations for each variable.

Folder "Input_Pertubed.zip" contains the perturbed Input data for the simulation model.
